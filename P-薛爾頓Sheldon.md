# 薛爾頓Sheldon

> 以胚胎學的觀點建立體型和性格間的關連。

|  類型 | 特徵                                             |
| :-: | :--------------------------------------------- |
| 內胚型 | 消化系統及臟器器官發達，成長後體型肥大圓軟，其特性為反應慢、喜交遊、待人寬容、情緒表達平衡。 |
| 中胚型 | 骨骼肌肉發達，成長後體型健壯魁梧，其特性為好冒險、喜運動、精力充沛。             |
| 外胚型 | 感覺器官與神經系統發達，成長後身軀瘦長，特性為善思考、易緊張、多愁善感            |
| 混合型 | 內中胚、外中胚型。上述各型不同比例的體型與特質呈現。                     |
