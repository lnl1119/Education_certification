# 精熟學習vs凱勒計畫

|      |       精熟學習      |         凱勒計畫        |
| :--: | :-------------: | :-----------------: |
| 熟練標準 |      80~90%     |         90%         |
| 學習進度 |       教師決定      |         學生決定        |
| 教學方式 |       團體學習      |         個別學習        |
| 單元測量 |  一同參加，自己(或教師)評分 |      個別參加、助理評分      |
| 校正技術 | 由標準答案得知可參加第二次測驗 | 由助理告知錯誤的地方,可參加無數次測驗 |
| 成績評量 |    期末總結性評量決定    |   單元考試佔75%，期末考佔25%  |
| 應用對象 |       中小學       |        大學或研究所       |
